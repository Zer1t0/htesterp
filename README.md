# HTesTerP

Check if http and https protocols are available for several hosts

## Installation

From crates.io (cargo):
```shell
cargo install htesterp
htesterp -h
```

From source:
```
git clone https://gitlab.com/Zer1t0/htesterp.git
cd ./htesterp
cargo build --release
./target/release/htesterp -h
```
